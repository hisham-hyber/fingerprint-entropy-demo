# Fingerprint Entropy (Critical Demo)

This is a simple interactive demo that estimates browser 
fingerprint entropy — and questions its validity.

## What it does
- Collects basic browser signals
- Estimates entropy (naively)
- Demonstrates why the result can be misleading

## Key idea
Entropy is often treated as a measure of uniqueness.  
This demo shows that this assumption is not always valid.

> Two users can have identical entropy — and be completely different.
